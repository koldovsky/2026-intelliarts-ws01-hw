# Proposal: Add keyboard shortcut for "Unlock All Elements"

## Problem
Currently, to unlock all elements, users must use the context menu or find the action in the UI. There is no keyboard shortcut for this common operation when one wants to quickly reset the canvas or edit previously locked items.

## Proposed Change
Add a keyboard shortcut `CtrlOrCmd+Shift+U` to trigger the `unlockAllElements` action.

## Impact
- Users can more quickly unlock all elements.
- Improved accessibility and productivity.
- Consistent with other shortcuts (e.g., `CtrlOrCmd+Shift+L` for toggle lock).
