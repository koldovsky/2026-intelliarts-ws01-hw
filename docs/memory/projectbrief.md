# Project Brief

Excalidraw is an open-source, browser-based whiteboard tool for creating hand-drawn-style diagrams. It is a large public TypeScript monorepo maintained by the Excalidraw team and a broad open-source community.

**Goal of this repo snapshot:** brownfield onboarding exercise for Intelliarts WS1. The README, dev-docs, and CLAUDE.md were intentionally removed so participants practice code archaeology.

**Core value proposition:** infinite canvas, real-time collaboration, rough/sketchy visual style, exportable to PNG/SVG/JSON.

**Audience:** developers building diagramming features or using `@excalidraw/excalidraw` as an embeddable React component.
