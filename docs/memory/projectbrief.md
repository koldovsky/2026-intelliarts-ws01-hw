# Project Brief

Excalidraw is a virtual whiteboard for sketching hand-drawn like diagrams. This repository is a snapshot of the Excalidraw project, structured as a monorepo.

## Goals
- Provide a collaborative, easy-to-use whiteboard experience.
- Support hand-drawn aesthetics for diagrams and sketches.
- Enable end-to-end encryption for collaborative sessions.
- Offer a highly extensible and customizable whiteboard component via the `@excalidraw/excalidraw` package.

## Core Features
- **Infinite Canvas:** Sketch freely without boundaries.
- **Hand-drawn look:** Unique visual style using Rough.js.
- **Collaboration:** Real-time collaboration with end-to-end encryption.
- **Export/Import:** Support for multiple formats including PNG, SVG, and `.excalidraw` files.
- **Extensibility:** Can be embedded into other React applications as a component.
- **Localization:** Support for multiple languages.
